Optional cancelToken

cancelToken?: CancelToken
  • Defined in convert.ts:141

Token that allows cancelling the PDF generation. All pending downloads will be terminated. The caller is responsible for removing underlying partial files and/or other user signaling.

Optional concurrency

concurrency?: number
  • Defined in convert.ts:134

Number of concurrent IIIF Image API requests to be performed, defaults to 1

Optional coverPageEndpoint

coverPageEndpoint?: string
  • Defined in convert.ts:151

Endpoint to contact for retrieving PDF data with one or more cover pages to insert before the canvas pages

Optional filterCanvases

filterCanvases?: readonly string[] | ((canvasId: string) => boolean)
  • Defined in convert.ts:121

Set of canvas ids to include in PDF, or a predicate to filter canvas identifiers by. By default, all canvases are included in the PDF.

Optional languagePreference

languagePreference?: string | readonly string[]
  • Defined in convert.ts:125

List of languages to use for metadata, page labels and table of contents, in descending order of preference. Will use the environment's locale settings by default.

Optional maxWidth

maxWidth?: number
  • Defined in convert.ts:130

Restrict the image size to include in the PDF. Only works with Level 2 Image API services that allow arbitrary downscaling, the conversion will not perform downscaling itself. For Level 1 endpoints, the closest available lower width will be selected.

Optional metadata

metadata?: { Author?: string; CreationDate?: Date; Keywords?: string; Title?: string }
  • Defined in convert.ts:143

Set PDF metadata, by default Title will be the manifest's label.

Optional ppi

ppi?: number
  • Defined in convert.ts:118

Pixels per inch to assume for the full resolution version of each canvas. If not set, the conversion will use an available IIIF Physical Dimensions service to calculate the page dimensions instead.

Optional preferLossless

preferLossless?: boolean
  • Defined in convert.ts:132

Prefer lossless formats (PNG or TIF) over lossy (JPG, default)
